# Fixed Income Risk Analytics

A Python project that performs bond valuation and fixed income risk analysis using discounted cash flow (DCF) techniques.

## Features

- Bond Pricing
- Macaulay Duration
- Modified Duration
- DV01
- Convexity
- Interest Rate Stress Testing

## Bonds Analysed

- 7.17 GS 2028
- 7.18 GS 2033
- 6.79 GS 2034

## Technologies

- Python
- Pandas

## Concepts Used

- Discounted Cash Flow
- Yield to Maturity
- Duration
- Modified Duration
- DV01
- Convexity
- Interest Rate Risk
- Stress Testing
